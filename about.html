Sobre o Roadmap

Esta aplicação foi desenvolvida para fins acadêmicos, durante a disciplina de Fundamentos de Programação Web, ofertada pela PUCPR no curso de Análise e Desenvolvimento de Sistemas.
A página, além de oferecer um roteiro e guia que facilita a compreensão das linguagens de programação, contempla tecnologias e aplicações em HTML, CSS, e Javacript.
O HTML foi usado para a estruturação das páginas, o CSS para estilização, e o Javascript para interatividade do conteúdo. Além disso, o Javascript foi aplicado para validar e recuperar os dados de um formulário via método GET.
